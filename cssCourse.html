Learn CSS

Start your journey to style your web pages with CSS!

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. It allows you to control the layout, colors, fonts, and other visual aspects of a webpage.

Basic CSS Syntax

CSS consists of rules, where each rule has a selector and a declaration block.

/* Example */
selector {
    property: value;
}

CSS Selectors

Selectors are used to target HTML elements. Here are some basic selectors:

  • Type Selector: Targets HTML tags (e.g., h1, p).
  • Class Selector: Targets elements with a specific class (e.g., .myClass).
  • ID Selector: Targets elements with a specific ID (e.g., #myId).

Adding CSS to HTML

You can add CSS to your HTML in three ways:

  • Inline CSS: Add styles directly within an HTML element using the style attribute.
  • Internal CSS: Add styles in a <style> block within the <head> of your HTML.
  • External CSS: Link to a CSS file using the <link> tag.

Common CSS Properties

Here are some commonly used CSS properties:

Property Description
color Sets the color of the text.
background-color Sets the background color of an element.
font-size Sets the size of the font.
margin Sets the space outside an element.
padding Sets the space inside an element.
